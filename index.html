Lua-libuv

Bem-vindo ao projeto Lua-libuv, um laboratório para explorar integração entre Lua e libuv.

Sobre o Projeto

O Lua-libuv é um ambiente para desenvolvimento e testes de scripts Lua com suporte a módulos em C, integrados com a biblioteca libuv para operações assíncronas e I/O não bloqueante.

Recursos

  • Suporte a módulos Lua compilados em C
  • Integração com libuv para operações assíncronas
  • Exemplos práticos de uso, como servidor HTTP
